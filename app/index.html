Мессенджер для команд, личных чатов, голосовых и звонков

MaxyMessenger объединяет сайт, десктопный запуск и Firebase-инфраструктуру в одном продукте. Быстрый чат, авторизация, голосовые сообщения и звонки работают в едином интерфейсе.

iPhone-ready

Сайт можно добавить на домашний экран iPhone как web app с полноэкранным запуском.

Telegram-похожий UI

Левая колонка чатов, верхняя панель диалога, пузырьки сообщений и быстрый composer.

Безопасность

Контроль доступа через Firebase

Аутентификация, правила базы и разделение данных пользователей строятся на Firebase. Это лучше, чем открытая база без auth, и дает основу для нормального продакшен-развития.

Что включено

  • Firebase Auth
  • Realtime Database
  • Структура `users`, `chats`, `messages`, `calls`
  • Сигналинг звонков в базе

Desktop

Запуск на компьютере

Сайт можно открывать как локальную desktop-версию через `start.bat`. Это удобно для ежедневной работы и тестов.

iPhone и App Store путь

Сейчас проект подготовлен как web app для iPhone и как база для дальнейшей упаковки в `.ipa` через Capacitor и Xcode на macOS.
